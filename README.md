# Indian-Railways-Delay-Analysis-using-SQL-Python-Power-BI
##Problem Statement
Train delays in Indian Railways affect service reliability and operational performance. This project aims to analyze delay patterns, identify high-risk trains and stations, measure on-time performance, and create an interactive Power BI dashboard using SQL and Python for actionable insights.

## Project Overview
This is an end-to-end data analytics project focused on analyzing train delays across Indian Railways.
The project uses **SQL** for querying and extracting insights, **Python** for data cleaning and preprocessing, and **Power BI** for building an interactive dashboard.
The dashboard helps track delay patterns, on-time performance, delay severity categories, and operational trends across trains and stations.

## Objectives

* Analyze train delay patterns across Indian Railways
* Identify the most delayed trains
* Identify stations with the highest delay occurrences
* Measure overall on-time performance
* Categorize delays into **On Time**, **Moderate Delay**, and **Heavy Delay**
* Build an interactive dashboard for operational performance monitoring

## Dataset

## Workflow

1. **SQL**

   * Queried and filtered train delay data
   * Performed aggregations for train-level and station-level insights
   * Extracted analysis-ready data for further processing

2. **Python**

   * Cleaned and preprocessed raw data
   * Handled missing values and formatting issues
   * Created delay categories for analysis
   * Prepared the final cleaned dataset for dashboarding

3. **Power BI**

   * Built an interactive one-page dashboard
   * Created KPI cards for key metrics
   * Added slicers for dynamic filtering
   * Designed visuals for delay trends, top delayed trains, top delayed stations, and operational performance

## Dashboard Features

* **KPI Cards**

  * Average Delay
  * Total Stations
  * Heavy Delay %
  * On-Time %

* **Slicers / Filters**

  * Train Name
  * Station Name
  * Delay Category

* **Visuals**

  * Top 10 Delayed Trains
  * Delay Category Distribution
  * Top 10 Delayed Stations
  * Train Delay Details Table
  * Operational Performance Breakdown

## Key Insights

* Certain trains consistently contribute to higher average delays
* Some stations experience significantly more delay occurrences than others
* Delay categorization provides a quick operational performance snapshot
* On-time percentage highlights the overall reliability of train operations
* Interactive filters allow deeper analysis by train, station, and delay category
